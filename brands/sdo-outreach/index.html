About the program

San Diego Opera's Outreach Program brings music fundamentals to student musicians across our community. These games turn practice into play — short rounds you can finish in minutes, built to be repeated until the skills stick.

Real piano, really recorded

Ear-training rounds use actual piano recordings — 36 chromatic notes and true harmonic context, not synth beeps.

Grows with the student

Short, repeatable rounds with levels that climb from first white-key finds and steady beats to full-key fluency and functional hearing.

Open under the hood

Built on Key Change, an open platform for community-made music games — so the library keeps growing.
